Tóm tắt về quảng bá trang tin

Thời gian thực hiện: 2 tiết

Để cài đặt Git và đăng ký tài khoản GitHub, bạn có thể làm theo các bước sau:

Bước 1: Cài đặt Git

1.1 Cài đặt Git trên Windows

  1. Tải Git: Truy cập trang web chính thức của Git: https://git-scm.com và nhấn vào nút "Download for Windows".

  2. Cài đặt Git:

    • Mở tệp vừa tải về và làm theo hướng dẫn của trình cài đặt.
    • Khi cài đặt, bạn có thể để mặc định tất cả các thiết lập, bao gồm việc tích hợp Git với command line và Git Bash.

  3. Kiểm tra cài đặt Git:

    • Sau khi cài đặt xong, mở Command Prompt (hoặc Git Bash) và nhập lệnh sau để kiểm tra phiên bản Git:
      git --version
    • Nếu Git được cài đặt thành công, bạn sẽ thấy phiên bản Git hiển thị.

1.2 Cài đặt Git trên macOS

  1. Cài đặt Git bằng Homebrew:

    • Nếu bạn đã cài Homebrew, chạy lệnh sau để cài đặt Git:
      brew install git

  2. Cài đặt Git từ Xcode:

    • Nếu chưa cài Homebrew, bạn có thể cài đặt Git bằng Xcode Command Line Tools:
      xcode-select --install

  3. Kiểm tra cài đặt Git:

    • Mở Terminal và chạy lệnh:
      git --version
    • Nếu Git đã được cài đặt, bạn sẽ thấy phiên bản Git.

1.3 Cài đặt Git trên Linux

  1. Cài đặt Git qua terminal:

    • Trên Ubuntu hoặc các bản phân phối Debian:
      sudo apt update
sudo apt install git
    • Trên Fedora:
      sudo dnf install git

  2. Kiểm tra cài đặt Git:

    • Mở terminal và chạy lệnh:
      git --version

Bước 2: Đăng ký tài khoản GitHub

  1. Truy cập GitHub: Vào trang web chính thức của GitHub: https://github.com.

  2. Tạo tài khoản:

    • Nhấn vào nút "Sign up" (Đăng ký).
    • Điền email, mật khẩu, và chọn tên người dùng (username) cho tài khoản GitHub của bạn.
    • Sau khi điền đầy đủ thông tin, nhấn "Create account" (Tạo tài khoản).

  3. Xác minh tài khoản:

    • Bạn sẽ nhận được một email xác nhận từ GitHub, mở email đó và làm theo hướng dẫn để kích hoạt tài khoản của bạn.

Bước 3: Cấu hình Git với tài khoản GitHub

  1. Thiết lập tên người dùng: Mở terminal hoặc Git Bash và chạy lệnh:

    git config --global user.name "Tên của bạn"

  2. Thiết lập email: Dùng địa chỉ email mà bạn đã đăng ký GitHub:

    git config --global user.email "email@example.com"

  3. Kiểm tra cấu hình:

    • Bạn có thể kiểm tra lại cấu hình của mình bằng lệnh:
      git config --list

Bước 4: Kết nối Git với GitHub

  1. Tạo SSH Key (khuyến nghị):

    • Tạo SSH key nếu bạn chưa có:
      ssh-keygen -t rsa -b 4096 -C "email@example.com"
    • Sau đó, thêm SSH key vào GitHub bằng cách copy nội dung SSH key và thêm vào GitHub qua trang SSH and GPG keys.

  2. Clone repository từ GitHub:

    • Bạn có thể clone một repository có sẵn hoặc tạo mới trên GitHub và clone về máy:
      git clone git@github.com:username/repo.git

Giờ bạn đã cài đặt Git, đăng ký tài khoản GitHub, và có thể bắt đầu quản lý mã nguồn của mình!

Để tạo và public (công khai) một thư mục trong GitHub, bạn có thể làm theo các bước sau:

Bước 1: Tạo repository trên GitHub

  1. Truy cập GitHub: Mở trang web GitHub và đăng nhập vào tài khoản của bạn.

  2. Tạo repository mới:

    • Nhấn vào biểu tượng dấu cộng (+) ở góc phải trên cùng và chọn "New repository".
    • Điền tên cho repository của bạn, ví dụ: my-repo.
    • Chọn chế độ Public để repository của bạn có thể được công khai (mọi người có thể xem).
    • (Tùy chọn) Thêm tệp README.md nếu bạn muốn.
    • Nhấn vào nút Create repository để hoàn tất việc tạo repository.

Bước 2: Tạo và public thư mục từ dòng lệnh Git

2.1 Clone repository về máy tính

Nếu bạn đã có repository trên GitHub, bạn cần clone nó về máy tính của mình để có thể thêm thư mục và tệp.

git clone https://github.com/username/my-repo.git

Thay username bằng tên GitHub của bạn và my-repo bằng tên repository của bạn.

2.2 Tạo thư mục trong repository

  1. Chuyển vào thư mục của repository:

    cd my-repo

  2. Tạo thư mục mới: Sử dụng lệnh mkdir để tạo thư mục mới trong repository của bạn:

    mkdir new-folder

    Thay new-folder bằng tên thư mục mà bạn muốn tạo.

  3. Thêm tệp vào thư mục: Thư mục chỉ được công khai khi nó có chứa ít nhất một tệp. Bạn có thể tạo một tệp trong thư mục đó, ví dụ như README.md hoặc một tệp khác:

    echo "# Nội dung thư mục" > new-folder/README.md

2.3 Thêm thư mục vào Git và commit

  1. Kiểm tra các thay đổi: Sử dụng lệnh git status để kiểm tra những thay đổi trong repository của bạn:

    git status

  2. Thêm thư mục vào Git: Thêm thư mục và các tệp liên quan vào Git:

    git add new-folder

  3. Commit các thay đổi: Commit các thay đổi với một thông điệp phù hợp:

    git commit -m "Add new folder"

2.4 Push thay đổi lên GitHub

Đẩy (push) các thay đổi từ máy tính của bạn lên GitHub:

git push origin main

Bước 3: Kiểm tra thư mục trên GitHub

Sau khi bạn đã thực hiện các bước trên, thư mục mới sẽ được tạo và public trong repository của bạn trên GitHub. Bạn có thể truy cập vào repository trên GitHub và kiểm tra xem thư mục có xuất hiện hay chưa.

Tùy chọn: Public repository đã có

Nếu bạn đã có một repository và muốn chuyển nó từ private (riêng tư) sang public (công khai), bạn có thể thực hiện như sau:

  1. Truy cập vào repository của bạn trên GitHub.
  2. Đi tới tab Settings.
  3. Cuộn xuống phần Danger Zone và chọn Change repository visibility.
  4. Chọn Public và xác nhận việc thay đổi.

Sau khi làm xong, repository và tất cả các thư mục trong đó sẽ trở thành công khai.
